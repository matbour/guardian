    # Methodology

    Benchmarks are located in the /benchmarks directory and achieved with the PHPBench library.

    They are run on GitHub Actions runners. Since all servers have different hardware, the values provided are only indicative.

    For each benchmark, we configured the runner to iterate 100 times on the tested feature excluding 10 "warmup interations".
